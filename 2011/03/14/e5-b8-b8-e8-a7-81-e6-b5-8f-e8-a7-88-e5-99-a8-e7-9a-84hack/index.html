常见浏览器的hack

2011-03-14

常见浏览器的hack ，ie 6,7,8;火狐，Chrome ，Safari，Opera.
[code=”css”]
/ hack for IEs/
.footwrap .plist{
padding-left:58px
padding-left:58\9; / 所有IE浏览器 /
padding-left:57px; /ie6,7/
_padding-left:56px; /ie6/
padding-left:55px\0; / ie8/
}
/ hack for ie9 /
body:not(:target) {
color: red\9; / IE9 */
}

/ hack for firefox/
@-moz-document url-prefix(){
.footwrap .plist{
padding-left:57px;
}
}

/ hack for Chrome and safari/
@media screen and (-webkit-min-device-pixel-ratio:0) {
.footwrap .plist{
padding-left:58px;
}
}

/ hack for opera /
@media all and (min-width:0px){
.footwrap .plist{
padding-left:58px;
}
}

[/code]
