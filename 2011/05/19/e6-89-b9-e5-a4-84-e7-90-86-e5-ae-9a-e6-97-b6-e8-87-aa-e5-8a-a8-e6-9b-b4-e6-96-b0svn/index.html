批处理定时自动更新svn

2011-05-19
下载

下载 后解压放到合适的文件夹，内部有两个 bat文件，autosvn.bat 和 setup.bat ,其中setup.bat 用于安装，可以右键，选择文本编辑器编辑这个文件，修改其中的时间，默认是晚上11点进行一次更新。
autosvn.bat 需要配置，

set svn_home=C:\Program Files\TortoiseSVN\bin
set svn_work=D:\wwwroot\static
set setup_path=D:\conf\autosvn

svn_home 是svn 的安装地址到，取到bin目录
svn_work 是你要更新的工程路径
setup_path 是你放置此安装文件 setup.bat 的路径

其中每次自动更新会写log。在你的setup_path 目录会生成一个 autosvn.log 文件
