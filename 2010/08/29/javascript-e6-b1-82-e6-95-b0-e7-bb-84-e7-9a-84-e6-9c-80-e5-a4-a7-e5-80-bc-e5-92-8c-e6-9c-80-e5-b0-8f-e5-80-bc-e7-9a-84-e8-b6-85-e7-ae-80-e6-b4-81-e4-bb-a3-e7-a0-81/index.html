JavaScript 求数组的最大值和最小值的超简洁代码

2010-08-29


How can we find the Min/Max number in an array?

看John Resig的Secrets of the JavaScript Ninja，被震了一下，原来Math.min和Math.max可以这么用。。



    function smallest(array){
      return Math.min.apply( Math, array );
    }
    function largest(array){
      return Math.max.apply( Math, array );
    }
    alert(smallest([0, 1, 2, 3]) == 0, “Locate the smallest value.”);
    alert(largest([0, 1, 2, 3]) == 3, “Locate the largest value.”);
