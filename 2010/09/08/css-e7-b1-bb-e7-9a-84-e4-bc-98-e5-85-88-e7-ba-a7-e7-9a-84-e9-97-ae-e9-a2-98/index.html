css 类的优先级的问题

2010-09-08

以前感觉自己很熟悉了，今天才发现一个缺口.问题如下：
我们定义两个类

那么“美拓blog” 是什么颜色？padding-left 是 20px 还是 100px； 在我印象里 应该根据类的顺序class=”b r”那么 r 类应该覆盖 b 类，即” 美拓blog” 应该是红色，padding-left是100px。
但这是错误的，实际上是什么样子的？css 类的优先级是根据定义的先后顺序 确定渲染时的优先级，因为 b 类在 r 类后定义，所以 b 类和 r 类同时 应用到同一个元素的时候，b 类将 覆盖 r 类。所以美拓blog是 蓝色，padding-left是100px。
感觉自己html，css都不熟，哈哈，记录一下。
