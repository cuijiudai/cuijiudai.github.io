jQuery 插件mtSlider 1.0 发布

2010-02-25


今天发布一个自己写的jquery插件mtslider，如图，就是一个简单的图片slider，看看 demo 吧

用法如下
[code=”javascript”]
$(“.slideImg”).mtSlider({
speed: 2000,
pause:2500,
direction:”right”
});
[/code]
有三个参数:
speed ：就是图片滚动的速度，
pause ：是一个图片完成一次滚动的时间，所以在设置是要使pause大于speed，
direction ：是滚动方向，有top，buttom，left，right可选

图片数量随便，但是不要多到超出slide的宽度，3-6个图片就可以了，另外就是右边的数字图片会自动添加，所以你就不用另外写html了,如果有兴趣可以下载使用,文章底部提供下载。
