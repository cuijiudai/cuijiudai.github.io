5个简单有用的css属性

2009-11-13

译自：webdesignerwall

1. CSS clip裁切属性

裁切像一个遮罩. 它能让你将目标内容遮罩成长方形. 必须的设置: 将目标物体设置成position:absolute. 然后依次设置clip的top,right,bottom,left值。

图片裁切例子(例子)

首先将父级div设置为position:relative,然后将<img\>设置为position: absolute,参见下图


[code=”css”]
.clip {
position: relative;
height: 130px;
width: 200px;
border: solid 1px #ccc;
}
.clip img {
position: absolute;
clip: rect(30px 165px 100px 30px);
}
[/code]

图片重设大小并裁切 (demo)

我想将原来的图片变为原来的 50% 来做一个缩略图.所以, 所以我用width和height来变换图片的大小并用clip来遮罩图像，然后使用 left属性使图像左移5px (博主注释：因为图像宽100px，左右各遮罩了15px，所以左移15px，使图像居中)


[code=”css”]
.gallery li {
float: left;
margin: 0 10px 0 0;
position: relative;
width: 70px;
height: 70px;
border: solid 1px #000;
}
.gallery img {
width: 100px;
height: 70px;
position: absolute;
clip: rect(0 85px 70px 15px);
left: -15px;
}
[/code]

2. Min-height (demo)

min-height （博主：这个大家应该很熟悉，不多说了）.


[code=”css”]
.with_minheight {
min-height: 550px;
}
[/code]

hack ie6

[code=”css”]
.with_minheight {
min-height:550px;
height:auto !important;
height:550px;
}

[/code]

解释：

1、由于IE6对!important的解释存在bug，所以它只识别height:500px这一句，超过500px后，IE6自身会增加盒子的高度；

2、IE7和其他标准浏览器三句都能识别，虽然定义height:500px，但我们又有了height:auto !important，所以超过500px也能自动增加盒子的高度。

3. White-space属性 (demo)

white-space 属性设置如何处理元素内的空白，值 pre-wrap 和 pre-line 是 CSS 2.1 中新增的，例如white-space: nowrap就规定段落中的文本不进行换行,详情见

[code=”css”]
em {
white-space: nowrap;
}

[/code]

4. Cursor属性 (demo)

如果你想更改Button的行为，你也应该更改cursor. 例如,当一个Button不可用的，你应该将cursor设置为default(arrow)来指出bottom禁用了，所以cursor属性对web app很有用。

[code=”css”].disabled {
cursor: default;
}

.busy {
cursor: wait;
}

.clickable:hover {
cursor: pointer;
}

[/code]

5. Display inline / block (demo)

块元素呈现在一个新行, 而inline元素会呈现在同一行. .

[code=”css”]
.block em {
display: block;
}

.inline h4, .inline p {
display: inline;
}

[/code]
