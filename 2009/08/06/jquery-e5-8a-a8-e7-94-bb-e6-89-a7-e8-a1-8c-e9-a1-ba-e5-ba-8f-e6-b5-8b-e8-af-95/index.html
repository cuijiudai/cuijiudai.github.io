Jquery动画执行顺序测试

2009-08-06

最近在学Jquery啊，但是始终没什么动力，也挺累了。今天看了一个Jquery的一个动画，没怎么看懂，特此测试总结。
HTML代码
[code=”html”]
<!DOCTYPE html PUBLIC “-//W3C//DTD XHTML 1.0 Transitional//EN”
“http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">


测试动画顺序











[/code]

CSS代码如图
[code=”css”]

#wrap{
height:500px;
width:500px;
border:1px solid blue;
position:relative;
}

#box1{
height:200px;
width:200px;
background:red;
position:absolute;
}

#box2{
height:200px;
width:200px;
background:green;
position:absolute;
top:200px;
}
[/code]
初试的js代码：
[code=’js’]
$(function(){
$(‘#button’).click(function(){
$(‘#box1’).animate({left:”200px”},3000)
.animate({top:”200px”},2000)
.fadeOut(1000);
})
})
[/code]

顺序执行，向左移动200px，向下200px，消失

[code=’js’]
$(function(){
$(‘#button’).click(function(){
$(‘#box1’).animate({left:”200px”},3000)
.animate({top:”200px”},2000)
.css(“border”,”5px solid yellow”);
})
})[/code]
.css不在动画队列中，率先立即执行

[code=’js’]
$(function(){
$(‘#button’).click(function(){
$(‘#box1’).animate({left:”200px”},3000)
.animate({top:”200px”},2000,function(){
$(this).css(“border”,”5px solid yellow”);
})

})

})[/code]
css加入到动画队列中，在前两个动作执行完后执行。
[code=’js’]
$(function(){
$(‘#button’).click(function(){
$(‘#box1’).animate({left:”200px”},3000)
.animate({top:”200px”},2000,showBorder);

    function showBorder(){
              $('#box1').css("border","5px solid yellow");
          }
})

})[/code]
顺序执行 ，注意showBorder的调用没有括号。
[code=’js’]
$(function(){
$(‘#button’).click(function(){
$(‘#box1’).animate({left:”200px”},3000)
.animate({top:”200px”},2000,showBorder());

    function showBorder(){
              $('#box1').css("border","5px solid yellow");
          }
})

})
[/code]
加上括号后，showBorder（）会立刻执行。
showBorder和showBoreder()是不同的。showBoreder()也会脱离动画队列立刻执行。
[code=’js’]
$(function(){
$(‘#button’).click(function(){
$(‘#box1’).animate({left:”200px”},3000)
.animate({top:”200px”},2000,showBorder())
.css(“background”,”pink”)

    function showBorder(){
              $('#box1').css("border","5px solid yellow");
          }
})

})
[/code]
如上代码，showBorder()和.css(“background”,”pink”)会立刻执行。然后动画执行。

[code=’js’]
$(function(){
$(‘#button’).click(function(){
$(‘#box1’).animate({left:”200px”},3000)
.animate({top:”200px”},2000,showBorder)
.css(“background”,”blue”)
.fadeOut(“fast”)

    function showBorder(){
              $('#box1').css("border","5px solid yellow");
          }
})

})
[/code]
看如上代码，先执行.css(“background”,”blue”)，然后动画animate({left:”200px”},3000)
.animate({top:”200px”},2000,showBorder)
执行,然后.fadeOut(“fast”)执行，可见动画队列没有被打破，依然顺序执行。

再改代码如下
[code=’js’]
$(function(){
$(‘#button’).click(function(){
$(‘#box1’).animate({left:”200px”},3000)
.animate({top:”200px”},2000,showBorder)
.css(“background”,”blue”)
$(“#box2”).fadeOut(“fast”)

    function showBorder(){
              $('#box1').css("border","5px solid yellow");
          }
})

})
[/code]
可见.css(“background”,”blue”)和 $(“#box2”).fadeOut(“fast”)会立刻执行。#box1然后动画执行
。即在句柄更换后，动画队列中#box1和#box2各自独立执行。
如果再加入#box1的动画
[code=’js’]
$(function(){
$(‘#button’).click(function(){
$(‘#box1’).animate({left:”200px”},3000)
.animate({top:”200px”},2000,showBorder)
$(“#box2”).fadeOut(“fast”)
$(“#box1”).animate({width:”100px”},2000)

    function showBorder(){
              $('#box1').css("border","5px solid yellow");
          }
})

})
[/code]
$(“#box1”).animate({width:”100px”},2000)会加入到box1 的动画中，在showBorder执行完后执行。

提高实例,再见啊！！
